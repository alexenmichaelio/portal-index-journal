# portal-index-journal

一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为静态页面的集合与索引。

## 项目简介

`portal-index-journal` 旨在提供一个轻量的 HTML 页面归档与展示空间。  
你可以在这里浏览一系列独立编写的页面，它们可能涵盖不同主题或用途，彼此之间没有直接关联。  
仓库本身不提供搜索、推荐或导航功能，仅通过目录结构组织页面。

## 目录结构

```
portal-index-journal/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面引用的资源文件（如样式、脚本、图片）
├── index.html      # 仓库首页（可选，用于列出页面链接）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件代表一个独立页面，文件名可反映内容类型或序号。
- `assets/`：按需存放页面所需的 CSS、JavaScript、图像等资源。
- `index.html`：提供一个简单的页面列表，方便访问所有归档页面（非必需）。

## 页面归档说明

- 所有页面均为纯 HTML 编写，不依赖后端服务。
- 页面之间相互独立，无数据交互或共享状态。
- 归档内容可能不定期更新或清理，以保持仓库整洁。
- 不保证所有页面长期可用或兼容所有浏览器。
